# Requisitos Funcionais – AuthFlow Web

## RF-01
O sistema deve permitir que usuários realizem login utilizando e-mail e senha válidos.

## RF-02
O sistema deve impedir login após 5 tentativas consecutivas com senha inválida.

## RF-03
O sistema deve exibir mensagens de erro claras e orientadas ao usuário.

## RF-04
O sistema deve permitir cadastro de novos usuários com e-mail único.
