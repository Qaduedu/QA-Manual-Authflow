# Bug – Falha ao adicionar item no carrinho

## Ambiente
- Navegador: Chrome
- Sistema Operacional: Windows 11

## Passos para Reproduzir
1. Acessar o site do e-commerce
2. Navegar até a página de um produto disponível
3. Clicar no botão “Add to Cart”

## Resultado Atual
O produto deve ser adicionado ao carrinho
O contador do carrinho deve ser atualizado

## Resultado Esperado
O produto não é adicionado ao carrinho
O contador do carrinho permanece inalterado
Nenhuma mensagem de erro ou confirmação é exibida

## Severidade
Crítica
O bug impede a ação principal do usuário (adicionar produto ao carrinho)
