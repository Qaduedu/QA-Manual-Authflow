# QA Manual – AuthFlow Web

Este repositório contém um projeto completo de **Testes Manuais** aplicado a um sistema web de autenticação (login e cadastro).

O objetivo é demonstrar:
- Pensamento crítico de QA
- Análise de requisitos
- Escrita de casos de teste
- Estratégia de testes
- Relato de bugs
- Relatórios de qualidade

Este projeto fundamenta e antecede a automação implementada em Cypress no repositório correspondente.
